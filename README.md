## Getting Started

Install node.js, npm, nextjs and truffle.

Clone this repo.

Copy .env.development to .env.

Install dependencies

```bash
npm install

```

Build the code

```bash
npx next build

```

Compile the contracts

```bash
tuffle compile

```

Run the server

```bash
npm run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
